# Rovo Automation - 26 Days Progress Report

Interactive investor presentation showcasing the progress of Rovo Automation in the first month (26 days).

## Features
- **Light Corporate Theme**: Professional, clean design with white background and blue accents.
- **Interactive Slides**: Keyboard and mouse navigation, animated counters, and transitions.
- **Client Engagement Map**: Visual representation of client outreach in North and West India.
- **Product Portfolio**: Interactive flip cards showing product details.
- **Responsive Design**: Adapts to different screen sizes.
